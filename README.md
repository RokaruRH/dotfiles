## Setup

> Brew - https://brew.sh

> Fish shell - https://fishshell.com

```
# Set as default shell
echo $(which fish) | sudo tee -a /etc/shells
chsh -s $(which fish)
```

> Starship prompt - https://starship.rs

> asdf to manage multiple runtime versions - https://asdf-vm.com `(fish + git)`
